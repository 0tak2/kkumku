## 정책

### Configs.plist

- Configs.plist에는 디버그용 설정, API 키 등 환경에 따라 다르게 적용될 필요가 있는 값들을 포함
- 디버그용 설정은 필요에 따라서 임시로 토글해서 활용하고, 커밋에는 포함하지 않도록 주의
- API Key와 같이 공개되어서는 안되는 값들은 환경변수로 포함
